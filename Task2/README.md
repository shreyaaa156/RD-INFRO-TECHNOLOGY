# Task 2 - Data Cleaning

## 📌 Objective

The objective of this project is to clean a raw dataset by handling missing values, duplicate records, and inconsistent data.

---

## 🛠️ Tools Used

* Python
* Pandas

---

## 📂 Dataset

The dataset contains customer-related information such as:

* Customer ID
* Age
* Gender
* City
* Salary
* Purchase Amount
* Membership Status
* Purchase Date

---

## ⚙️ Steps Performed

1. Imported dataset using Pandas
2. Checked missing values using `isnull()`
3. Filled numerical missing values using **mean**
4. Filled categorical missing values using **mode**
5. Removed duplicate rows
6. Cleaned column names (lowercase & removed spaces)
7. Saved cleaned dataset

---

## 📊 Results

* All missing values handled successfully
* Duplicate rows removed
* Dataset is now clean and ready for analysis

---

## 📁 Files Included

* `data.csv` → Raw dataset
* `cleaned_data.csv` → Cleaned dataset
* `cleaning.py` → Python script

---

## 🚀 Conclusion

This project demonstrates basic data cleaning techniques which are essential for data analysis and machine learning workflows.
