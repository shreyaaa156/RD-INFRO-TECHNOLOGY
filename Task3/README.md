# Task 3 - Data Transformation

## 📌 Objective

The objective of this task is to transform the cleaned dataset into a structured and analysis-ready format by performing data type conversions and feature engineering.

---

## 🛠️ Tools Used

* Python
* Pandas

---

## 📂 Dataset

The dataset used in this task is the cleaned dataset obtained from Task 2 (`cleaned_data.csv`), which contains customer-related information.

---

## ⚙️ Steps Performed

1. Loaded the cleaned dataset using Pandas
2. Converted the `Purchase_Date` column into datetime format
3. Created a new feature called **Age_Group** based on age values:

   * Young (< 30)
   * Middle Age (30–49)
   * Senior (50+)
4. Prepared the dataset for further analysis
5. Saved the transformed dataset

---

## 📊 Output

A new dataset `transformed_data.csv` was generated with additional features and proper data formatting.

---

## 📁 Files Included

* `transformation.py` → Python script for transformation
* `transformed_data.csv` → Transformed dataset
* `cleaned_data.csv` → Input dataset from Task 2

---

## 🚀 Outcome

The dataset is now properly structured and enriched with new features, making it ready for exploratory data analysis and visualization.
